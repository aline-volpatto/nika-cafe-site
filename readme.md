 # Nika Café — Website (Projeto de Estudo)

Este repositório contém o desenvolvimento de um website fictício criado como parte do meu processo de aprendizado em **desenvolvimento front-end**.

O **Nika Café** é um conceito de cafeteria imaginária criado exclusivamente para fins de estudo. O objetivo do projeto é praticar a construção de páginas web estáticas, organização de layout e criação de interfaces responsivas utilizando tecnologias fundamentais da web.

---

## Objetivo do projeto

Este projeto foi desenvolvido para exercitar conceitos essenciais de desenvolvimento front-end, incluindo:

* Estruturação semântica de páginas com **HTML**
* Estilização e organização visual com **CSS**
* Criação de **layouts responsivos**
* Uso de **Flexbox** para posicionamento de elementos
* Organização de arquivos em um pequeno projeto web

Além disso, o projeto busca simular a construção de um site institucional simples para um negócio local.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (interações básicas)
* Flexbox
* Media Queries (responsividade)

---

## Funcionalidades implementadas

* Página inicial apresentando o conceito da cafeteria
* Seção sobre métodos de preparo do café
* Página de acompanhamentos sugeridos
* Formulário de contato
* Navegação entre páginas
* Layout adaptável para diferentes tamanhos de tela

---

## Responsividade

O layout foi desenvolvido para se adaptar a diferentes resoluções de tela, proporcionando uma experiência adequada em:

* desktops
* tablets
* dispositivos móveis

---

## Observação

Este projeto **não representa um negócio real**.
Todo o conteúdo, marca e conceito apresentados foram criados apenas para fins educacionais e prática de desenvolvimento web.
